# MusicPlayer-RebileAdarsh
Arduino Music Player by Rebile Adarsh. Portable MP3 player built on Arduino Nano with DFPlayer Mini, SH1106 OLED display, and 5-button control. Plays tracks from SD card, shows track info &amp; progress bar. Features: Play/Pause, Next/Prev, Shuffle, Repeat modes. Robot eye splash animation on boot. Safe boot flag prevents crashes.
