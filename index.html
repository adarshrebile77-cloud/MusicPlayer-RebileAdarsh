Capabilities

Key Features

Play / Pause
First press plays Track 1. Subsequent presses toggle playback state via DFPlayer hardware command.
Shuffle Mode
Next/Prev picks a random track. Avoids replaying the same track twice consecutively.
Repeat Mode
Loops current track forever using DFPlayer's hardware loop — zero CPU overhead.
OLED Display
SH1106 128×64 in page-buffer mode. Track number, play state, shuffle/repeat and progress bar.
Safe Boot Flag
dfOK guard prevents garbage serial commands when DFPlayer absent. No fake PLAYING state.
Eye Animation
Robot eyes shift left, right, then blink on boot, followed by a full system checklist screen.
